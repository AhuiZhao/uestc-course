# 数学实验

电子科技大学数学科学学院教授张勇、房秀芬、张晓伟等《数学实验》课程。课程的内容包括：MATLAB语言程序设计基础，符号计算与微积分实验，线性代数实验，数值计算实验，最优化模型实验，随机模拟实验。其特点是以MATLAB语言程序设计为基础，通过应用案例增强学生对数学方法、数学模型的认识，培养数学实验思维、数学建模思维。

期末上机考试题目为上机测试题原题随机抽取（可能有改编）。
